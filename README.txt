# Apartman TKON web stranica

Ovo je gotova mini web stranica za QR kod za Apartman TKON – 6 osoba.

## Što je unutra
- index.html
- style.css
- assets/ fotografije apartmana
- QR kod: assets/qr_apartman_tkon.png

QR je trenutno napravljen za ovu adresu:
https://alen-sekula.github.io/apartman-tkon/

Ako ti je GitHub korisničko ime ili naziv repozitorija drugačiji, QR treba ponovno napraviti za stvarnu adresu.

## Kako objaviti na GitHub Pages

1. Na GitHubu napravi novi repository:
   apartman-tkon

2. Uploadaj sve datoteke iz ovog ZIP-a:
   - index.html
   - style.css
   - assets folder

3. Otvori repository Settings

4. Idi na Pages

5. Source stavi:
   Deploy from a branch

6. Branch:
   main / root

7. Nakon 1-2 minute stranica bi trebala biti na:
   https://alen-sekula.github.io/apartman-tkon/
